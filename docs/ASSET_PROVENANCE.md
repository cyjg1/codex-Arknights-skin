# Asset provenance

## Operator backgrounds

The following images were generated for Codex Arknights Skin using OpenAI's built-in image generation tool on 2026-07-16. They are project-bound fan-theme assets, not official game artwork.

| Asset | Character | Visual brief |
|---|---|---|
| `operator-amiya.png` | Amiya | Rhodes Island corridor, cool daylight, cyan tactical palette |
| `operator-kaltsit.png` | Kal'tsit | Medical research bay, clinical light, mint and charcoal palette |
| `operator-chen.png` | Ch'en | Rainy Lungmen checkpoint, blue-hour light, red signal accents |
| `operator-texas.png` | Texas | Logistics depot at dusk, graphite and violet palette |
| `operator-exusiai.png` | Exusiai | Lungmen rooftop, warm daylight, coral and sky-blue palette |
| `operator-w.png` | W | Ruined industrial district at night, carbon and deep-red palette |

Shared production constraints: original 16:9 anime game key visual; character on the right; quiet negative space on the left for Codex UI; recognizable public character traits; no copied composition; no text, logo, watermark, or embedded interface.

The final assets are stored in both `macos/assets/` and `windows/assets/` so each platform package remains self-contained. The corresponding files are byte-identical across platforms.

## README screenshots

`docs/screenshots/preview-night.png`, `preview-day.png`, and `preview-controls.png` were captured from `docs/preview.html` at 1280×720. The preview loads the production CSS, theme manifest, and bundled operator assets, but uses repository-authored demonstration copy only. No live Codex task, account, directory, or conversation data appears in these images.
